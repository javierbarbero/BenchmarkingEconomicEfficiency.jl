Profit Russell model

The profit Russell model is computed by solving a graph Russell DEA model for the technical efficiency.

In this example we compute the profit efficiency Russell measure under variable returns to scale:

julia> X = [2; 4; 8; 12; 6; 14; 14; 9.412];

julia> Y = [1; 5; 8; 9; 3; 7; 9; 2.353];

julia> W = [1; 1; 1; 1; 1; 1; 1; 1];

julia> P = [2; 2; 2; 2; 2; 2; 2; 2];

julia> profitrussell = deaprofitrussell(X, Y, W, P)

deaprofitrussell Function Documentation

BenchmarkingEconomicEfficiency.deaprofitrussell — Function
deaprofitrussell(X, Y, W, P)

Compute profit efficiency using Russell data envelopment analysis for inputs X, outputs Y, price of inputs W, and price of outputs P.

Optional Arguments

  • names: a vector of strings with the names of the decision making units.

Examples

julia> X = [2; 4; 8; 12; 6; 14; 14; 9.412];

julia> Y = [1; 5; 8; 9; 3; 7; 9; 2.353];

julia> W = [1; 1; 1; 1; 1; 1; 1; 1];

julia> P = [2; 2; 2; 2; 2; 2; 2; 2];

julia> deaprofitrussell(X, Y, W, P)
